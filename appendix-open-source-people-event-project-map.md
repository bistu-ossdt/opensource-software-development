# 附录 开源人物-事件-项目对照表

本附录是一张检索用导航表，不承担完整历史叙述，也不承担人物展开介绍。它的作用只有一个：当读者先记住了某个人物、某个事件或某个项目时，能迅速把这三个入口重新连到一起，并决定下一步应回到哪一部分继续阅读。

因此，本附录与后两份附录的分工是明确的：

- 想按时间理解历史主线，读“附录 开源历史关键事件时间线”
- 想按人物理解持续贡献，读“附录 开源人物谱”
- 想快速查找“人物 - 事件 - 项目/组织”的对应关系，先看本附录

## 4. 使用方式

这份对照表最适合三种检索方式：

- 先记得人物，再反查他对应的关键事件与项目
- 先记得事件，再回到人物与组织理解其位置
- 先记得项目或组织，再快速定位它在全书中的历史与治理位置

如果读者想看“线性的历史叙述”，应继续阅读“附录 开源历史关键事件时间线”；如果读者想看“人物维度的持续贡献”，应继续阅读“附录 开源人物谱”。
